# USACO Guide

A training and a helper for one to keep implementing by hand in order not to forget how to type. 

There is an ease of using newer methods, new paradigms for the sake of speed but biological systems sacrifice their rigorous thinking over those in the long run. That doesn't mean that we have to go back to abacus or sheep counting but we need to know and be ready for difficult times.

So, this codebase keeps one rigorous, and help one express clearly at the cost of speed.
