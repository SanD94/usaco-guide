# USACO Guide

A training and a helper for one to keep implementing by hand in order not to forget how to type. 

 The main concern is that there is an ease of using newer methods, new paradigms for the sake of speed (i.e., LLMs) but biological systems sacrifice their rigorous thinking over those in the long run. That doesn't mean that we have to go back to abacus or sheep counting but we need to know and be ready for difficult times.

So, this codebase keeps one rigorous, and help one express themselves clearly at the cost of speed.

The guide has levels to complete. In each level, it consists of different types of algorithms which can be implemented by hand with the pseudo-code provided in different topic. For the implemenation of those, there are practice problems to solve.
